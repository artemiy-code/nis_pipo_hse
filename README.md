## API Endpoints

### Authentication
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход и получение JWT
### Items
Все endpoints требуют аутентификации.
- `POST /api/items` - создание нового объекта
- `GET /api/items/{id}` - получить объект по id
- `GET /api/items` - получение списка всех объектов у текущего пользователя
- `PUT /api/items/{id}` - обновить уже созданный объект
- `DELETE /api/items/{id}` - "Мягкое удаление" объекта

### Быстрый старт
1. Создать БД:
```azure
CREATE DATABASE appdb;
```
2. В `application.yml` изменить или подогнать под:
```
url: jdbc:postgresql://localhost:5432/appdb
username: postgres
password: Artem
```
Сами таблицы будут созданы автоматически с помощью Flyway при старте приложения

___
- URL: http://localhost:8080
- Swagger ui: http://localhost:8080/swagger-ui.html
- Метрики: http://localhost:8080/actuator/metrics

Из них кастомные бизнесс-метрики:
-  `items.created.total`
- `items.updated.total`
- `items.deleted.total`

Приложение пишет логи в `logs/application.log`
